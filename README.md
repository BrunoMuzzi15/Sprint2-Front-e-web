📱 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de apresentar uma nova funcionalidade fictícia para os celulares da empresa JOVI chamada Luz Dinâmica.

A proposta consiste em um sistema inteligente de iluminação integrado ao smartphone, capaz de adaptar cores, brilho e efeitos luminosos de acordo com notificações, músicas, jogos, chamadas e ambiente do usuário.

Além da criação da ideia do produto, também foi desenvolvida uma página web utilizando:

HTML
CSS
JavaScript

A página apresenta informações sobre a funcionalidade, imagens conceituais do projeto e elementos visuais interativos.

   Objetivo

O principal objetivo do trabalho foi:

Criar uma proposta inovadora para smartphones
Desenvolver uma interface moderna e interativa
Aplicar conceitos de front-end
Trabalhar estilização e interatividade com JavaScript

   Funcionalidades da Página

Página responsiva
Efeitos hover em elementos clicáveis
Exibição de imagens do projeto
Seções informativas sobre o modo Luz Dinâmica

 Sobre o Modo Luz Dinâmica

O modo Luz Dinâmica foi pensado para melhorar a experiência do usuário através de iluminação inteligente no celular que ao tirar uma foto em um ambiente de festa automaticamente é feita uma adaptação melhorando a qualidade da foto.

Buscamos apresentar uma interface tecnológica e moderna, simulando uma apresentação real de um novo recurso para smartphones da JOVI.
